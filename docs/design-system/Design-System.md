# Design System: RichLegend AI SaaS Starter Kit 디자인 시스템

## 1. 목적

이 문서는 제품 UI를 만들 때 따라야 할 시각 언어, 컴포넌트, 상태, 접근성 기준을 정의합니다. 새 화면이나 컴포넌트를 만들기 전 이 문서에 요구사항과 디자인 결정을 먼저 적습니다.

## 2. 디자인 원칙

- 업무형 SaaS에 맞게 조용하고 명확한 화면을 우선합니다.
- 마케팅 영역은 설명력이 있어야 하지만, 앱/대시보드는 정보 스캔과 반복 사용성을 우선합니다.
- 카드 남용을 피하고, 반복 항목/모달/도구 영역에만 카드 스타일을 사용합니다.
- 버튼, 폼, 메뉴, 탭, 다이얼로그는 Base UI primitive와 TailwindCSS token을 기준으로 만듭니다.
- 모든 색상과 간격은 `app/globals.css`, `tailwind.config.ts`, 이 문서의 토큰을 기준으로 관리합니다.

## 3. 토큰

| Token | 용도 |
| --- | --- |
| `--background` | 전체 배경 |
| `--foreground` | 본문 텍스트 |
| `--muted` | 보조 배경 |
| `--muted-foreground` | 보조 텍스트 |
| `--border` | 경계선 |
| `--primary` | 주요 액션 |
| `--accent` | 성공/강조 |
| `--danger` | 위험/오류 |

## 4. 컴포넌트 기준

| 컴포넌트 | 사용 기준 |
| --- | --- |
| Button | 명확한 명령에만 사용하고, 주요 CTA는 하나만 강조합니다. |
| Link | 페이지 이동에는 Link를 사용합니다. |
| Dialog | 확인, 설정, 짧은 폼 입력에 사용합니다. |
| Card | 반복 정보 항목, 요금제, 요약 패널에 사용합니다. |
| Badge | 상태값을 짧게 표시할 때 사용합니다. |
| Table | 결제 이력, 사용자 목록, 감사 로그처럼 비교가 필요한 데이터에 사용합니다. |

## 5. 상태

- Loading: 레이아웃 전체가 아니라 콘텐츠 영역에만 skeleton을 둡니다.
- Empty: 다음 행동을 함께 제공합니다.
- Error: 사용자가 할 수 있는 복구 행동을 포함합니다.
- Disabled: 이유가 불명확하면 도움말 또는 설명을 붙입니다.
- Success: 짧고 구체적으로 결과를 알려줍니다.

## 6. 접근성

- 버튼과 링크의 의미를 섞지 않습니다.
- 아이콘 버튼에는 `aria-label`을 제공합니다.
- 제목 계층은 `h1 → h2 → h3` 순서를 유지합니다.
- 폼 입력은 label과 error message를 연결합니다.
- 키보드만으로 dialog, menu, tab을 조작할 수 있어야 합니다.

## 7. 새 화면 작성 전 체크

- [ ] 이 화면의 요구사항이 `docs/requirements/PRD.md`에 있습니다.
- [ ] 사용자 흐름이 `docs/requirements/Use-Case.md`에 있습니다.
- [ ] 라우팅과 정보 구조가 `docs/requirements/IA.md`에 있습니다.
- [ ] 데이터가 필요한 화면이면 `docs/requirements/ERD.md`에 테이블과 관계가 있습니다.
- [ ] API/DB/권한 요구사항이 `docs/requirements/TRD.md`에 있습니다.
- [ ] 사용 컴포넌트와 상태가 이 디자인 시스템에 맞습니다.

## 8. 다국어 UI 기준

- 기본 화면 언어는 한국어이며, 영어는 상단 언어 선택 메뉴에서 전환합니다.
- 20개 국어 이상 확장을 고려해 버튼, 탭, 카드, 내비게이션은 번역 길이가 길어져도 깨지지 않게 설계합니다.
- UI 문자열은 하드코딩하지 않고 locale별 copy helper 또는 메시지 파일에서 관리합니다.
- 언어별 날짜, 숫자, 통화, 가격, 법무 문구가 다른 경우 컴포넌트 요구사항에 명시합니다.
- 언어 선택기는 현재 locale을 명확히 표시하고, 같은 페이지의 다른 locale 경로로 이동해야 합니다.
- 새 화면 검증에는 한국어, 영어, 긴 번역 문자열이 들어간 샘플 locale을 포함합니다.
